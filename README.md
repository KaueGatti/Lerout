# Lerout

> Sistema PDV/ERP para rotisserias

## 🚀 Sobre o projeto

Um sistema desenvolvido para a realização e gerenciamento de pedidos de uma rotisseria.

## ✨ Funcionalidades

- Realização e Impressão de pedidos.
- Gerenciamento de pedidos, marmitas, produtos, bairros, clientes, motoboys e cardápio.
- Visualização de relatório diário.

## 🛠️ Tecnologias

- JavaFX
- SQLite
- Git

## 📋 Pré-requisitos

- Java >= 25.0.0
